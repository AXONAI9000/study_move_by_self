# Day 06: 泛型编程与能力系统深入

## 📚 学习目标

今天你将学习：
- 掌握 Move 泛型编程的核心概念
- 深入理解四种能力（Abilities）：copy, drop, store, key
- 学会在实际项目中正确使用泛型和能力
- 理解能力约束对代码安全性的影响

## 🎯 为什么重要

泛型和能力系统是 Move 语言最独特的特性：
- **泛型**：让代码可复用，避免重复
- **能力系统**：在编译期保证资源安全，这是 Move 能防止双花等问题的核心机制
- 在 DeFi 开发中，你会频繁使用泛型来处理不同类型的代币

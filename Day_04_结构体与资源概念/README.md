# Day 04 - 结构体与资源概念

## 学习目标
- 掌握struct结构体的定义与使用
- 理解Move的资源（Resource）概念
- 学习四大能力：copy、drop、store、key
- 了解资源的线性类型特性

## 核心内容
1. 结构体定义：struct关键字、字段类型
2. 结构体实例化与字段访问
3. 能力声明：has copy, has drop, has store, has key
4. 资源的唯一性与不可复制性
5. 引用与借用：&、&mut
6. 结构体的所有权转移

## 为什么重要？
资源是Move语言的核心创新，理解资源模型是掌握Move编程的关键，它确保了数字资产的安全性。

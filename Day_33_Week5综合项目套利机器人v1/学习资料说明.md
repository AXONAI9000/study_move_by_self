## ⚠️ 注意事项

### 安全提醒
1. **私钥管理**: 永远不要将私钥提交到代码仓库
2. **测试先行**: 在测试网充分测试后再考虑主网
3. **金额控制**: 设置合理的交易限额
4. **风险意识**: 了解套利的风险和竞争

### 常见问题

**Q: 编译失败怎么办？**
A: 检查 Move.toml 配置，确保依赖版本正确。

**Q: 测试网连接不上？**
A: 可能是网络问题，尝试更换节点或使用代理。

**Q: 如何调试 Move 合约？**
A: 使用 `aptos move test` 运行测试，查看详细输出。

**Q: WebSocket 连接失败？**
A: 检查 DEX 的 API 文档，确认 WebSocket URL 正确。
